# React Native 

### Package Install

```
brew install node

brew install watchman

brew install flow


// 설치하는 과정에서 문제가 있다면 brew 업그레이드 해준다.

brew update

brew upgrade

```


### React Native Install

```
npm install -g -react-native-cli
```


### New Application 생성

```
react-native init 프로젝트이름
```

### Xcode 실행하기

```
react-native open 프로젝트이름.xcodeproj

```

### Simulator 실행하기
```
react-native run-ios

```


### 실행에 실패 했을 때

```
cd 해당 디렉토리

npm install

npm start

```
